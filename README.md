# isPalindrome
------------------------
解题思路：
把整数除以10取余，求出整数的每一位数，然后放到数组里，然后从两边往中间去遍历整个数组，判断该整数是否是回文；
---------------------------

判断一个整数是否是回文数。回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。

示例 1:

输入: 121
输出: true
示例 2:

输入: -121
输出: false
解释: 从左向右读, 为 -121 。 从右向左读, 为 121- 。因此它不是一个回文数。
示例 3:

输入: 10
输出: false
解释: 从右向左读, 为 01 。因此它不是一个回文数。
